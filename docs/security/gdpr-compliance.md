# GDPR Compliance Strategy

## Principles

1.  **Lawfulness, Fairness, and Transparency:**
    - Inform users about data collection (Privacy Policy).
    - Collect only necessary data.

2.  **Purpose Limitation:**
    - Use data only for invoicing and business operations.

3.  **Data Minimization:**
    - Do not store excessive personal details.

4.  **Accuracy:**
    - Allow users to update their profile.

5.  **Storage Limitation:**
    - Define retention periods (e.g., 7 years for financial records).

6.  **Integrity and Confidentiality:**
    - Use encryption and secure access controls.

## Rights of Data Subjects
- Right to Access
- Right to Rectification
- Right to Erasure ("Right to be forgotten") - *Note: Limited by tax law requirements.*
- Right to Data Portability (Export function).
