# Tech Stack

## Core Technology
| Component | Technology | Version | Rationale |
| --- | --- | --- | --- |
| **Framework** | Next.js | 14.x | Server Components, SEO, Routing |
| **Language** | TypeScript | 5.x | Type safety, maintainability |
| **Styling** | TailwindCSS | 3.x | Utility-first, rapid UI development |
| **UI Lib** | shadcn/ui | Latest | Accessible, customizable components |
| **Database** | PostgreSQL | 15+ | Relational data integrity, complex queries |
| **ORM** | Prisma | 5.x | Type-safe DB access, migrations |
| **Deployment** | Railway | - | PaaS, easy setup, built-in DB |
| **Auth** | NextAuth.js | 5.x | Flexible auth for Next.js |
| **AI** | OpenAI API | GPT-4o | Natural language processing |

## Compatibility Matrix

- **Node.js:** >= 20.x
- **PostgreSQL:** >= 15.x
- **Next.js:** 14.x (App Router)

## Infrastructure
- **Hosting:** Railway
- **CI/CD:** GitHub Actions
- **Storage:** Blob Storage (via Railway plugin or external S3 if needed later)
